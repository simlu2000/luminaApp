☀️ Lumina

Lumina is a minimalist Progressive Web Application (PWA) designed for photographers. It calculates precise Golden Hour and Blue Hour timings based on real-time geolocation and provides smart Creative Advice by cross-referencing weather data with solar positioning.

✨ Key Features
📍 Precision Tracking: Real-time solar data calculation via Geolocation API.

📊 Solar Path Visualization: Custom polar coordinate chart showing sun elevation and light phases (built with Apache ECharts).

🌦️ Photographer's Weather: Focuses on metrics that matter: Cloud Cover, Visibility, and UV Index.

💡 Creative Engine: Dynamic logic that generates photography tips based on current atmospheric conditions.

📱 iOS-Style UI: A premium Glassmorphism interface designed with a "mobile-first" approach.

🛠️ Tech Stack
Frontend: React.js (Functional Components, Hooks, Context API).

Data Viz: Apache ECharts (Polar Coordinate Systems).

APIs: OpenWeatherMap API for real-time meteorological data.

PWA: Service Workers and Web App Manifest for an installable, native-like experience.

Design: Planned in Figma using a modular 8pt grid and iOS-inspired aesthetics.

📈 Architecture Highlights

Clean Code: Modular component architecture and custom hooks for API management.

UX/UI Focus: Implementation of complex design trends like Glassmorphism and "Soft Shadows" with high performance.

Data Interpretation: Transforming raw JSON data from external APIs into actionable, human-readable creative insights.
